2.12 As Supresas de declarações com VAR

  • Declarações com var não criam escopo de bloco assim como em outras liguagens.
  • Com var, podmeos declarar variáveis com o mesmo nome sem que isso incomode o interpretador do JavaScript

2.13 As declarações de variáveis com LET

  • Escopo de bloco - variáveis declaradas com let dentro de um bloco só são acessíveis dentro desse bloco
  • Não é possível haver declarações repetidas dentreo de um mesmo bloco, ou seja, não podemos declarar duas variávies com o mesmo nome dentro de um mesmo bloco.

3 - No Cangaço é ação para todo lado.

3.1 O pepel de um um controlador

  • No modelo MVC, o Controller faz a ponte de ligação entre as ações do usuário na view com o Model. Na view temos nosso client/index.html e no Model temos instâncias de Negociação. Dessa forma, matendo a separação entre o Model e View, alteração na View não afetam o Model.
